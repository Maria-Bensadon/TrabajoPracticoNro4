# TrabajoPracticoNro4
Trabajo Practico N°4 - Implementación de un Servidor con Express y Arquitectura MVC

###

## Descripción
En este trabajo práctico, se implementa un servidor con Node.js y Express que seguirá la arquitectura
Modelo-Vista-Controlador (MVC). El servidor se ejecutará en el puerto 3005 y escuchará diversas
peticiones GET.

## Tecnologías
Node.js v20+ (módulos ES con .mjs)
Express v4 (libreria utilizada)
NPM (gestiona las librerias del proyecto)

## Requisitos previos
Node.js >= 18 instalado en tu máquina
NPM >= 9 (se instala junto con Node.js)
